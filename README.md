# PhiloQuest: Семинар 2

Игровой тренажёр для подготовки к устному семинару по философии: объяснение материала, задания, слабые темы, быстрый бой и финальный босс.

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Где редактировать контент

Основной игровой контент находится в `src/data/gameData.ts`.

## Проверки

- Coverage: `src/utils/validators.ts`, экран `CoverageCheckScreen`.
- Data check: `src/utils/validators.ts`, экран `DataCheckScreen`.
- localStorage: `src/utils/storage.ts`.
